# Frontend Wizards Stage 1b - Testable Profile Card

This project is the HNG Frontend Stage 1b task implementation: Accessible and responsive profile card built with vanilla HTML, CSS, and JavaScript.

## Profile Subject

This profile card is for **Aang**, the main character from *Avatar: The Last Airbender*.

## Required Test IDs Included

- `test-profile-card`
- `test-user-name`
- `test-user-bio`
- `test-user-time`
- `test-user-avatar`
- `test-user-social-links`
- `test-user-hobbies`
- `test-user-dislikes`

## Features

- Semantic structure using `article`, `header`, `figure`, `nav`, and `section`
- Keyboard-focusable social links with visible focus states
- Live-updating epoch time in milliseconds (`Date.now()`) every second
- Responsive layout:
  - Mobile: stacked
  - Tablet/Desktop: avatar left, content right
- Aang avatar image rendered via URL with meaningful `alt` text

## Run Locally

Open `index.html` directly, or serve the folder:

```bash
npx serve .
```
